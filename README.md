# ChatApp — WhatsApp Clone 💬

Aplicación de chat en tiempo real con interfaz estilo WhatsApp.

## Características
- 🔐 Login y Registro con JWT
- 💬 Chat en tiempo real con Socket.io
- 📷 Envío de imágenes
- 🎤 Notas de voz
- 👥 Lista de miembros con estado online/offline
- 💾 Persistencia de datos (JSON)
- 🌐 Multi-usuario (cualquier dispositivo en la red)

## Requisitos
- Node.js 18+

## Instalación

### Backend
```bash
cd server
npm install
npm start
```

### Frontend
```bash
cd client
npm install
npm run dev
```

## Acceso
- **Local:** http://localhost:3000
- **Red:** http://TU_IP:3000 (para conectarte desde otra computadora)

Para obtener tu IP: ejecuta `ipconfig` en Windows.
